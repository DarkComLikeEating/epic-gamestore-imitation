# epic-gamestore-imitation

大一下学期期末的校企合作培训项目，当时教了H5，CSS，BootStrap v3 + jQuery的响应式页面。结课作业是模仿现有的网页使用H5自己写一份页面，我们小组选择了Epic游戏商城。

模仿页面有三个，分别是游戏商品页面，免费游戏页面和愿望清单页面。
